# Proyecto de Navidad de Unity de Informática Gráfica

Se ha creado un juego móvil usando Unit,y de varios minijuegos, basados en reflejos. Se ha usado android para desarrollarlo.

## Proyecto

Subida las carpetas de los Assets donde se han creado las escenas y la carpeta de los scripts con varios por escena.
